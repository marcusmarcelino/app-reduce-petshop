## Informações Gerais
Projeto em fase de desenvolvimento, sistema de gerenciamento para Petshop.

## Instalação
Para que o projeto funcione corretamente você deverá clonar o projeto e inicializá-lo executando:
```bash
yarn install
yarn start
```

## Backend
